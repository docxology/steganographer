🔐 What's Being Signed
Encode: PCM audio samples → BLAKE3 hash (32 B) → Ed25519 signature (64 B) → 104-byte payload embedded in LSB of audio samples using keyed PRNG index selection.
Decode: Extract LSB payload from audio → deserialize frame_index + hash + signature → verify signature matches the extracted hash.
Round-trip: If the decoded hash + signature match, the audio chunk has not been tampered with since signing.
